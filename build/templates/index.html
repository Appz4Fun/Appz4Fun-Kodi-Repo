Appz4Fun Kodi Repository

Pick a channel, download its repository add-on, and install it in Kodi via Add-ons → Install from zip file. Kodi will then auto-update along that channel.

{channels}

How to install

  1. Settings → System → Add-ons → enable Unknown sources.
  2. Add-ons → Install from zip file → choose the channel zip above.
  3. Add-ons → Install from repository → Appz4Fun Repository → install add-ons.
